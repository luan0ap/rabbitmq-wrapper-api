    • RabbitMQ-API

    RabbitMQ.js provides a minimal higher-level wrapper around rabbitmq's API.

    Usage

    /*
  Data retrieved from API will return a raw axios
*/
const RabbitMQ = require('rabbitmq-api');

const rabbitMQ = RabbitMQ({
  hostname: 'http://localhost',
  username: 'guest',
  password: 'guest',
  vhost: '/'
})

rabbitMQ.queues.list()
  .then((data) => {
    // do some stuff
  })

rabbitMQ.queues.name('queue-name').get({} // axios params are allowed)
  .then((data) => {
    // do some stuff
  })

    API Documentation

    [API documentation][docs] is hosted on github pages, and is generated from JSDoc; any contributions should include updated JSDoc.

    Installation

    RabbitMQ.js is available from npm or yarn.

    npm install https://github.com/luan0ap/rabbitmq-wrapper-api

    yarn add https://github.com/luan0ap/rabbitmq-wrapper-api

    Tests

    The main way we write code for rabbitmq-api is using test-driven development. We use Jest to run our tests. Given that the bulk of this library is just interacting with RabbitMQ's API, nearly all of our tests are integration tests.

    To run the test suite, run npm run test.
